# Tienda E-commerce Salahogar

Salahogar es una tienda ecommerce donde puedes ver una lista de productos, visualizar detalles de cada producto, agregar las cantidades deseadas a tu carrito de compras y realizar una orden. Los usuarios pueden completar un formulario con sus datos para finalizar la orden y recibir un ID de Orden. 

## Características

- Ver una lista de productos
- Visualizar detalles de productos
- Agregar productos al carrito de compras
- Ver carrito de compras
- Realizar una orden
- Completar un formulario para finalizar la orden
- Recibir un ID de Orden
